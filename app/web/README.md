# app/web

Это smoke/debug слой backend-приложения.

Что здесь допустимо:
- smoke player
- debug pages
- ручная проверка streaming / metadata / API

Что здесь НЕ является целью:
- production frontend
- полноценный клиентский интерфейс
- личные кабинеты
- крупная frontend-логика

Основной frontend должен жить отдельно в `/frontend`.
